# SmartTest

Platforma do rozwiązywania testów online z podziałem na role użytkowników.

## Autorzy
**Jakub Wołczyński**

**Wiktor Kieć**


## Funkcjonalności

- **Admin** – tworzenie i zarządzanie testami, klasami, pytaniami oraz użytkownikami
- **Uczeń** – rozwiązywanie testów w określonym czasie, podgląd wyników
- **Limit czasowy** – każdy test ma określony czas na rozwiązanie
- **Wyniki** – automatyczne sprawdzanie odpowiedzi i generowanie wyników po zakończeniu testu

## Technologie

- Java, Spring Boot
- PostgreSQL
- React

## Uruchomienie

### Backend

`cd backend`

`mvn spring-boot:run`

### Frontend

`cd frontend`

`npm install` #Jeżeli nie ma npm

`npm start`

## Szybki start

**Windows:** uruchom `start.bat`  
**Linux/Mac:** uruchom `./start.sh`
